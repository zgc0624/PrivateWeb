# 实用工具箱网站 - 设计文档

日期：2026-07-05

## 1. 项目概述

一个公开可访问的在线实用工具箱网站，用户无需注册登录即可直接使用所有工具。所有工具功能纯前端实现，不依赖后端服务，部署在免费静态托管平台，24小时可访问，无需本地电脑常开。

**首发工具（4个）：**
- 二维码生成与解析
- 图片处理（压缩、格式转换、尺寸调整）
- 单位换算
- 日期计算

## 2. 技术选型

| 类别 | 技术选型 | 说明 |
|------|---------|------|
| 构建工具 | Vite | 现代前端构建工具，启动快、热更新快 |
| 框架 | Vue 3 (Composition API) | 用户熟悉的前端框架 |
| UI组件库 | Element Plus | 配合Vue 3使用，组件丰富精致 |
| 路由 | Vue Router 4 | 单页应用路由，支持URL直达工具 |
| 二维码生成 | qrcode (npm包) | 成熟的二维码生成库 |
| 二维码解析 | jsQR (npm包) | 浏览器端二维码识别 |
| 图片处理 | 浏览器原生 Canvas API | 无需额外库，前端直接处理 |
| 部署平台 | Vercel | 免费、自动HTTPS、全球CDN、Git自动部署 |

## 4. 页面设计

### 4.1 全局布局
- **顶部导航栏**：左侧Logo+站点名称，右侧深色模式切换按钮；毛玻璃半透明效果，滚动时有样式过渡
- **主内容区**：根据路由渲染对应页面，留白充足
- **页脚**：简洁版权信息

### 4.2 首页
- **Hero区**：渐变背景，大标题"实用工具箱"，副标题"常用工具，即开即用"，带轻微装饰元素
- **搜索栏**：支持工具名称搜索过滤
- **工具卡片网格**：响应式布局（桌面4列→平板2列→手机1列）
  - 每个卡片：彩色SVG图标、工具名称、一句话简介
  - 微交互：hover时卡片上浮、阴影加深、轻微缩放
  - 点击跳转到对应工具页面 `/tools/{tool-id}`

### 4.3 工具页面通用设计
- 返回首页的面包屑/返回按钮
- 工具标题 + 简短说明
- 操作区和结果区分明，布局清晰
- 输入控件使用Element Plus组件，操作按钮样式统一
- 处理过程显示加载动画，结果淡入展示

## 5. 工具功能详情

### 5.1 二维码工具 (`/tools/qrcode`)
**Tab1 - 生成二维码：**
- 输入框：文本/网址输入
- 可选设置：尺寸（128/256/512/1024）、容错级别（L/M/Q/H）、前景色/背景色选择
- 快捷模板按钮：WiFi二维码、联系人(vCard)、短信、邮件
- 实时预览生成的二维码
- 下载按钮：支持PNG/SVG格式下载

**Tab2 - 解析二维码：**
- 图片上传区域（支持拖拽上传、点击选择、粘贴图片）
- 解析结果展示在文本框中，支持复制
- 解析失败时友好提示"无法识别二维码，请上传清晰图片"

### 5.2 图片处理 (`/tools/image-process`)
**功能：**
- 图片上传：支持拖拽、多选、批量处理
- 功能Tab：图片压缩、格式转换、尺寸调整
- 压缩：质量滑块（0-100），显示原始大小 vs 预计压缩后大小
- 格式转换：支持 JPG ↔ PNG ↔ WebP 互转
- 尺寸调整：按像素（宽x高）或百分比缩放，可选保持宽高比
- 处理前后对比预览（可切换并排/滑块对比）
- 单张下载/批量打包下载
- 大文件提示，避免浏览器卡顿

### 5.3 单位换算 (`/tools/unit-converter`)
- 分类标签切换：长度、重量、温度、面积、体积、速度
- 左右双面板：各带一个数值输入框和单位下拉选择器
- 实时双向换算：输入任意一侧，另一侧自动计算结果
- 常用单位置顶（如长度：米、千米、厘米、毫米、英寸、英尺、英里）
- 结果支持一键复制

### 5.4 日期计算 (`/tools/date-calc`)
**Tab1 - 日期差计算：**
- 两个日期选择器
- 显示结果：相差 X 天 / X 周 X 天 / X 个月 X 天 / X 年 X 个月 X 天
- 显示两个日期分别是星期几

**Tab2 - 日期推算：**
- 起始日期选择器（默认今天）
- 加/减切换、数字输入框、单位选择（天/周/月/年）
- 实时显示目标日期和星期几
- 快捷按钮：一周后、一个月后、本月底、今年底

## 6. 视觉设计风格

- **配色**：主色调采用清爽靛蓝色（#4F46E5），浅灰背景，深色文本；支持浅色/深色模式切换并持久化
- **圆角**：卡片12px圆角，按钮8px圆角，输入框统一圆角
- **阴影**：分层阴影体系，卡片默认柔和阴影，hover加深
- **动画**：所有交互（hover、点击、切换、结果展示）都有200-300ms平滑过渡
- **图标**：使用简洁线性SVG彩色图标，每个工具配专属配色图标
- **字体**：系统无衬线字体栈，中文优先，标题字重600，正文字重400
- **响应式**：完全适配手机、平板、桌面设备

## 7. 数据与隐私

- 所有工具运算完全在浏览器内存中执行，**不上传任何用户数据到服务器**
- 用户偏好（深色/浅色模式、最近使用工具）保存在 `localStorage`，仅本地存储
- 不使用Cookie，不做用户追踪，不收集任何统计数据

## 8. 错误处理

- 输入实时验证，非法输入即时提示（非阻塞式，用输入框状态提示）
- 图片上传：校验文件类型和大小，不支持的格式提前告知
- 二维码解析失败：友好提示而非报错
- 全局错误捕获：单个工具出错不影响整个站点
- 所有错误提示使用Element Plus Message组件，简洁友好

## 9. 开发与部署流程

**本地开发：**
1. `npm install` 安装依赖
2. `npm run dev` 启动本地开发服务器（默认 http://localhost:5173，支持热更新）
3. `npm run build` 构建生产版本到 `dist/` 目录
4. `npm run preview` 本地预览生产构建结果

**公开部署（Vercel）：**
1. 在GitHub创建仓库，推送代码
2. Vercel导入GitHub仓库，自动识别Vite项目
3. 每次push到main分支，自动构建部署，分配 `xxx.vercel.app` 域名（自带HTTPS）
4. 部署后网站24小时在线，无需本地电脑开机
5. 可选：绑定自定义域名

## 10. 不做的事情（YAGNI原则）

- 不做用户注册/登录系统
- 不做后端服务和数据库
- 不做服务端数据存储
- 不做用户数据同步
- 不做评论/社交功能
- 首发只做4个工具，后续工具增量添加
