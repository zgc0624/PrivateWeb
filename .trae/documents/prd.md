# 实用工具箱 - 产品需求文档 (PRD)

## 1. 产品概述
一个公开可访问的在线实用工具箱网站，用户无需注册登录，打开浏览器即可使用所有工具。首发4个高频日常工具：二维码生成解析、图片处理、单位换算、日期计算。
- 目标用户：日常需要使用小工具的普通用户和轻度开发者
- 核心价值：即开即用、无需安装、功能实用、界面美观、数据隐私（所有处理在浏览器本地完成）

## 2. 核心功能

### 2.1 功能模块
1. **首页**：Hero宣传区、工具搜索、工具卡片网格展示
2. **二维码工具页**：二维码生成（含快捷模板）、二维码解析
3. **图片处理页**：图片压缩、格式转换、尺寸调整
4. **单位换算页**：多分类单位实时双向换算
5. **日期计算页**：日期差计算、日期推算

## 3. 核心流程

用户访问网站 → 首页展示所有工具卡片 → 用户点击目标工具卡片或搜索过滤 → 进入工具页 → 使用工具功能（输入/上传→处理→查看结果→下载/复制）→ 可返回首页选择其他工具。

```mermaid
    "B" --> "C{搜索或选择工具}"

    "E" --> "F[浏览器本地处理]"

```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：靛蓝色系（#6366F1 为主色，#818CF8 为辅助色，#4F46E5 为深色），搭配渐变效果
- **强调色**：珊瑚橙（#F97316）用于CTA按钮和关键操作
- **按钮风格**：圆角（10px）、主色渐变填充、hover时轻微上浮+发光效果、secondary按钮为描边式
- **卡片风格**：16px圆角、柔和多层阴影、hover上浮8px+阴影加深+轻微缩放
- **字体**：中文使用系统无衬线字体栈，标题字重600-700，正文字重400，辅助文字字重400
- **布局风格**：桌面端居中容器（max-width: 1200px），顶部导航+内容区+页脚，卡片网格布局
- **图标风格**：Lucide风格线性图标，每个工具配专属彩色渐变图标
- **背景**：浅色模式下淡灰底+柔和渐变装饰光斑；深色模式下深色底+微光效果
- **动效**：页面元素入场渐显动画、卡片hover微交互、结果淡入、平滑滚动、深色模式切换过渡

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | Hero区 | 渐变背景、大标题（48px）、副标题、搜索框（圆角+阴影）、装饰性模糊色块 |
| 首页 | 工具卡片 | 彩色渐变图标区、工具名（18px bold）、简介（14px）、hover上浮+阴影发光 |
| 工具页 | 标题区 | 返回按钮、工具名、工具简介、分隔线 |
| 工具页 | 操作面板 | 卡片式面板包裹、分区清晰、输入控件统一风格 |
| 导航栏 | 全局 | 毛玻璃半透明、左侧Logo+名称、右侧深色模式切换按钮 |
| 二维码 | 预览区 | 白色卡片展示二维码、下载按钮组 |
| 图片处理 | 对比预览 | 处理前后并排对比、滑块对比切换 |

### 4.3 响应式设计
- 桌面优先设计，适配手机/平板/桌面
- 断点：sm(640px)、md(768px)、lg(1024px)、xl(1280px)
- 手机端：卡片单列、操作区纵向排列、导航栏简化
- 触摸优化：按钮最小44px点击区域、输入框适合触摸输入
